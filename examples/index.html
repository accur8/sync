Accur8 Sync Code Examples

These examples demonstrate common use cases and patterns for using Accur8 Sync.

BasicDatabaseSync.scala

Shows how to synchronize data between two databases with different database systems (PostgreSQL to MySQL).

Key concepts: Database configuration, connection factories, RowSync, sync results

StreamingLargeDataset.scala

Demonstrates memory-efficient processing of large datasets using streaming APIs.

Key concepts: Streaming queries, batch processing, memory management

JsonApiIntegration.scala

Shows how to integrate with external JSON APIs and sync the data to a database.

Key concepts: JSON codecs, API clients, async operations, error handling

DataValidationExample.scala

Illustrates data validation and cleansing during sync operations.

Key concepts: Validation rules, data transformation, error reporting

Running the Examples

To run these examples in your project:

1. Add Accur8 Sync dependencies to your build.sbt:
   libraryDependencies ++= Seq(
     "io.accur8" %% "a8-sync-shared" % "latest-version",
     "io.accur8" %% "a8-sync-api" % "latest-version"
   )

2. Copy the example code to your project

3. Update the database configurations to match your environment

4. Run with: sbt "runMain examples.BasicDatabaseSync"
